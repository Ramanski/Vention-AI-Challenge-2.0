# AI Challenge Task 1 Report

## Approach

I used an AI-enhanced IDE together with prompting techniques to generate the initial React application structure, leaderboard components, filters, sorting logic, and responsive layout.

After generating the first version, I iteratively refined the UI by comparing it with the original leaderboard and improving spacing, colors, typography, alignment, and overall appearance. Browser developer tools (F12) were used to inspect styles and dimensions. I also fixed several issues related to filtering and sorting logic during implementation.

## Tools and Technologies

- VS Code with GitHub Copilot
- Prompting techniques
- React 19
- React DOM 19
- TypeScript
- Vite 8
- Tailwind CSS 4
- PostCSS
- ESLint
- GitHub Pages (deployment)

## Data Replacement and Responsible AI Usage

To comply with responsible AI usage policies, no real corporate or personal data was used. All leaderboard entries were replaced with fictional mock data, including fake names, departments, job titles, scores, and avatar initials.

In some cases, sanitized screenshots with removed avatars and anonymized user information were used only as visual references for UI refinement.
